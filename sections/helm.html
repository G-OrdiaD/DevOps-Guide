4. Create Helm Chart

Helm Helm is a package manager for Kubernetes that helps you define, install, and upgrade complex Kubernetes applications. is used to manage Kubernetes applications. A Helm "chart" is a package of pre-configured Kubernetes resources. First, create the basic chart structure.

helm create chat-app-chart

Then, modify the generated files to define your application's deployment (`deployment.yaml`), how it's exposed (`service.yaml`), and its configuration (`values.yaml`).

values.yaml

replicaCount: 1 image: repository: chat-app tag: latest pullPolicy: IfNotPresent service: type: NodePort port: 3000 targetPort: 3000

deployment.yaml

apiVersion: apps/v1 kind: Deployment metadata: name: {{`{{ include "chat-app-chart.fullname" . }}`}}-deployment spec: replicas: {{`{{ .Values.replicaCount }}`}} template: spec: containers: - name: {{`{{ .Chart.Name }}`}} image: "{{`{{ .Values.image.repository }}`}}:{{`{{ .Values.image.tag }}`}}" ports: - containerPort: {{`{{ .Values.service.targetPort }}`}}

service.yaml

apiVersion: v1 kind: Service metadata: name: {{`{{ include "chat-app-chart.fullname" . }}`}}-service spec: type: {{`{{ .Values.service.type }}`}} ports: - port: {{`{{ .Values.service.port }}`}} targetPort: {{`{{ .Values.service.targetPort }}`}} selector: app.kubernetes.io/name: {{`{{ include "chat-app-chart.name" . }}`}}
