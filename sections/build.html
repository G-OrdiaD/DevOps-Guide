2. Build and Test Locally

Once the `Dockerfile` is created, you can build the image and run it as a container on your local machine to verify that it works correctly before moving to deployment.

Build the Docker Image

This command builds the image from your `Dockerfile` and tags it as `chat-app:latest`.

docker build -t chat-app:latest .

Run the Container Locally

This command runs the container, mapping port 3000 on your local machine to port 3000 inside the container. You can then access the app at `http://localhost:3000`.

docker run -p 3000:3000 chat-app:latest
