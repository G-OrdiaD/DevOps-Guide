1. Dockerize Application

The first step is to containerize the application by creating a `Dockerfile`. This file contains instructions to assemble a Docker image. We use a multi-stage build to create a lean, production-ready image, and a `.dockerignore` file to exclude unnecessary files.

.dockerignore

This file prevents local artifacts and dependencies from being copied into the image, keeping it small and secure.

node_modules npm-debug.log .env Dockerfile .git .gitignore

Dockerfile

This file defines a two-stage build process. The first stage builds the app, and the second creates a minimal final image.

# Stage 1: Build a lean production image FROM node:18-alpine AS builder WORKDIR /app COPY package*.json ./ RUN npm install COPY . . # Stage 2: Final production image FROM node:18-alpine WORKDIR /app COPY --from=builder /app ./ EXPOSE 3000 CMD ["npm", "start"]
