3. Setup Local Kubernetes

We'll use Docker Desktop's built-in Kubernetes cluster, which is perfect for local development. It provides a single-node cluster that's easy to set up and manage.

Enable Kubernetes in Docker Desktop

  1. Open the Docker Desktop dashboard and go to Settings.
  2. Navigate to the Kubernetes tab.
  3. Check the "Enable Kubernetes" box.
  4. Click "Apply & Restart".

After it starts, verify the setup by running this command in your terminal:

kubectl get nodes

You should see a single node named `docker-desktop` with a `Ready` status.
